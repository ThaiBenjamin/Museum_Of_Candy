# 🍬 Museum of Candy

A visually fun practice website styled with Bootstrap — built as a CSS/Bootstrap exercise to learn responsive multi-section page layouts with alternating image-text rows.

## 📚 What I Was Learning

Using Bootstrap to build a polished, responsive multi-section webpage — including navbar styling, full-height hero sections, alternating row layouts, and custom color overrides on top of Bootstrap's defaults.

---

## 🔑 Key Things Practiced

- **Bootstrap Grid** — Using row/col system to align image and text side-by-side
- **Navbar** — Fixed top navigation with Bootstrap classes
- **Section Layout** — Alternating left-right content blocks with background colors
- **Custom CSS** — Overriding Bootstrap defaults with a custom `app.css`
- **Responsive Design** — Layout that adapts to different screen sizes out of the box

---

## 💡 What It Taught Me

This project showed me how powerful Bootstrap is for quickly assembling a professional-looking page. More importantly, it taught me the right mental model for Bootstrap: start with the grid structure, then layer in Bootstrap utilities, then add custom CSS only where you need to deviate. The alternating section pattern (image left / text right, text left / image right) is a layout I've seen on countless real marketing pages — and now I understand exactly how it's built.
